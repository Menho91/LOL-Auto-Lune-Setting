# LOL-Auto-Lune-Setting
- 리그 오브 레전드 룬 자동 세팅

## ✅설명
- 실행한 후 챔피언 이름과 게임 모드를 입력하면 마우스를 움직여 알맞은 룬을 세팅합니다.

## 📚출처
- op.gg 에서 챔피언과 게임 모드에 적합한 룬 정보를 불러옵니다.

## ⛏문제점
- 클라이언트 창을 옮기면 마우스가 이동하는 위치와 어긋납니다.
- 능력치 파편 부분은 현재 자동으로 설정되지 않습니다.
